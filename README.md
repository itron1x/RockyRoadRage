## Description
Rocky Road Rage is a fun, split-screen Multiplayer game about racing rocks

## How to play
1) Download the latest release.
2) Install game via installer.
3) Execute the RockyRoadRage.exe file.

## Controls
<b>Mouse:</b>
* Movement - WASD
* Look around - mouse
* Jump - Space 
* Sprint - Shift
* Pause - P

<b>Controller:</b>
* Movement - Left joystick
* Look around - right joystick
* Jump - A
* Spring - Right shoulder
* Pause - Menu


## Compability
This game currently is PC only. However it does support controllers.
